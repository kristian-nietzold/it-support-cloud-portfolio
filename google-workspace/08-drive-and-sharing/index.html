Google Drive sharing controls, collaboration settings and retained project evidence.

Status: Archived and partially evidenced. The original Google Workspace subscription is no longer active, so this page presents only the configuration supported by retained screenshots.

08. Drive & Sharing

Configured Google Drive collaboration and file-sharing controls for the organisation.

The objective is to enable collaboration while maintaining control over access and data exposure.

Drive Access & Sharing

Drive sharing settings were configured to allow internal collaboration while controlling external access.

Configuration includes:

  • Internal sharing: Enabled

  • External sharing: Allowed (controlled)

  • Link sharing: Restricted to intended users

This ensures:

  • Secure collaboration within the organisation

  • Controlled external sharing

  • Reduced risk of accidental data exposure

Evidence Limitation

Shared Drive and group-based permission configuration formed part of the original project work. However, the supporting screenshots were not retained before the Google Workspace subscription ended.

Portfolio note: No replacement or third-party screenshots have been used. The page therefore distinguishes between the retained evidence above and the wider design approach documented below.

The intended collaboration model used:

  • Team-owned storage rather than dependence on an individual user account

  • Group-based access wherever practical

  • Restricted external sharing based on business need

Design Approach

The configuration follows:

  • Least privilege access

  • Group-based permission control

  • Separation of internal and external sharing

This ensures security while maintaining usability.

Summary

Retained evidence confirms:

  • Configuration of organisation-level Drive sharing settings

  • Controlled internal and external collaboration settings

  • Application of secure sharing and least-privilege principles

Shared Drive and detailed permission screenshots are not presented as verified evidence because they were not retained before access to the original tenant ended.
