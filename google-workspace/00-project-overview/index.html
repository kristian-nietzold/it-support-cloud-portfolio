Overview of the Google Workspace support lab and business scenario.

Context: This Google Workspace lab extends the same SaaS administration environment across identity, access, collaboration and security controls.

00. Project Overview

Introduction

This project documents the design and implementation of a Google Workspace environment for the fictional organisation Nietz Tech Ltd.

The goal of this lab is to replicate real-world administrative tasks performed by a Google Workspace Administrator, including user management, organisational structuring, security configuration, and policy enforcement.

Objectives

The objectives of this project are to:

  • Configure a custom domain within Google Workspace

  • Design a scalable organisational unit (OU) structure

  • Create and manage users and groups

  • Implement role-based access control (RBAC)

  • Configure core security settings (MFA, sharing controls, etc.)

  • Apply data protection policies (DLP, alerts, audit logging)

  • Demonstrate administrative best practices

Environment Overview

  • Organisation Name: Nietz Tech Ltd

  • Primary Domain: nietz.co.uk

  • Platform: Google Workspace

  • Admin Console: admin.google.com

This environment simulates a small-to-medium business with multiple departments and role-based access requirements.

Design Principles

The environment was designed using the following principles:

  • Least Privilege Access – Users only have access to what they need

  • Separation of Duties – Admin roles are isolated from standard users

  • Scalability – Structure supports business growth

  • Security First – Policies prioritise data protection and risk reduction

  • Centralised Control with Flexibility – OU-based inheritance with overrides

Outcome

The completed work covers:

  • Deploy and configure a Google Workspace environment

  • Implement structured user and access management

  • Apply enterprise-level security concepts

  • Document administrative processes clearly

It serves as both a learning project and a portfolio piece to showcase practical Google Workspace administration skills.
