# IT Support and Cloud Administration Portfolio

Practical portfolio evidence covering Windows Server, Active Directory,
Microsoft 365, Microsoft Entra ID, hybrid identity, Microsoft Intune,
ServiceNow-based support investigations and Google Workspace administration.

The environment follows the simulated company Nietz Ltd through infrastructure
deployment, cloud administration, endpoint management and structured Service
Desk troubleshooting.

Networking work is identified separately as planned and is not presented as
completed evidence.
