Microsoft Infrastructure, Microsoft 365 and Cloud Administration

A staged business environment following Nietz Ltd from on-premises Active Directory into Microsoft 365, Microsoft Entra ID, hybrid identity and Microsoft Intune endpoint management.

Technical Evidence

This portfolio is built as a continuing business environment rather than a collection of isolated screenshots. Each phase adds practical evidence relevant to Microsoft support, managed service provider operations, cloud administration and technical troubleshooting.

Windows and Active Directory Support

Domain controller deployment, DNS, domain join, Active Directory users and groups, file shares, permissions and Group Policy.

Microsoft 365 Support

Cloud users, licensing, Exchange Online, Outlook, Teams, SharePoint and OneDrive administration and collaboration support.

Identity and Endpoint Support

Microsoft Entra administration, delegated roles, MFA, hybrid identity, Intune enrolment, compliance, application deployment and Windows Update management.

Development Roadmap

Future portfolio development will extend the same Nietz Ltd environment into additional Microsoft security, collaboration, Azure and automation workloads. These areas will be added as public phases once completed evidence is available.

Security, Compliance and Governance

Microsoft Defender, Microsoft Purview, Compliance Manager, sensitivity labels, retention, data loss prevention and identity governance.

Microsoft 365 Services and Collaboration

Deeper Exchange Online, Teams, SharePoint, OneDrive, guest access, external sharing and service administration.

Azure Infrastructure and Automation

Azure networking, virtual machines, storage, RBAC, policy, monitoring, backup and administrative automation with PowerShell and Microsoft Graph.
